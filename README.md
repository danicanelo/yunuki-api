## Descripción

API para la aplicación Yunuki.

## Instalación

```bash
$ npm install
```

## Lanzar la aplicación

```bash
$ npm run start
```

## Modo desarrollo (reinicio automático tras cualquier cambio)

```bash
$ npm run start:dev
```
